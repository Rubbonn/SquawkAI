# SquawkAI
SquawkAI è un assistente RAG che analizza carte aeronautiche, meteo e regolamenti PDF per pianificare il tuo volo. Risponde a query tecniche, interpreta le procedure nazionali e traccia rotte dinamiche su mappe interattive. Un vero copilota digitale che trasforma manuali complessi in un dialogo fluido e preciso. Clear for takeoff!
